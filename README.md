# 📖 Overview

Hookora is an AI-powered workflow platform designed to help creators, marketers, freelancers, founders, and businesses produce better content in less time.

Instead of relying on generic AI chatbots, Hookora provides specialized AI tools built around real content workflows.

Users can generate:

- 🎯 Viral Hooks
- ✍️ Captions
- 📰 Titles
- 💡 Content Ideas
- 🧠 Marketing Copy
- 📱 Social Media Posts
- 🏷️ Hashtags
- 📈 Content Strategies
- 🎬 Creator Workflows

The platform combines prompt engineering, AI orchestration, reusable workflows, and scalable SaaS architecture to streamline modern content creation.

# 🚀 Getting Started

## Clone Repository

```bash
git clone https://github.com/Mikadorey/Hookora.git
```

---

## Install Dependencies

```bash
pip install -r requirements.txt
```

---

## Environment Variables

Create a `.env` file.

```env
DATABASE_URL=

OPENAI_API_KEY=

JWT_SECRET_KEY=

PAYSTACK_SECRET_KEY=

EMAIL_USERNAME=

EMAIL_PASSWORD=
```

---

## Run Development Server

```bash
uvicorn app.main:app --reload
```
